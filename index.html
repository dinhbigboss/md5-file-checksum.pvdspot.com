FAQs

Is it safe for sensitive files?

Absolutely. Your file is never uploaded to any server. All calculations happen within your browser's memory using the FileReader API.

Can MD5 be reversed?

MD5 is a one-way cryptographic hash function. While not "reversible," weak passwords can be cracked using rainbow tables. It is best used for integrity checks, not encryption.

What is a "Collision"?

A collision occurs when two different files produce the same MD5 hash. While rare, MD5 is no longer collision-resistant, which is why SHA-256 is preferred for high-security needs.

What is the file size limit?

The limit depends on your browser's RAM. Most modern browsers can handle files up to 2GB. For larger files, the page might become unresponsive during the calculation.
